🧮 Simple Calculator

📌 Objective

The objective of this project is to create a simple calculator that performs basic arithmetic operations such as addition, subtraction, multiplication, and division using web technologies.

🛠️ Tools Used

HTML – To structure the calculator layout
CSS – To design and style the calculator interface
JavaScript – To implement calculation logic and user interaction
GitHub – For version control and project hosting

🔧 Steps Performed

Created the calculator structure using HTML.
Styled the calculator buttons and display using CSS.
Implemented arithmetic operations using JavaScript.
Added event handling for button clicks.
Tested the calculator for correct functionality.
Uploaded the project to GitHub.

✅ Outcome

A fully functional simple calculator web application that allows users to perform basic calculations with a clean and user-friendly interface.
